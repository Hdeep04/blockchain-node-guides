# blockchain-node-guides

## 概要
50代キャリアチェンジ・ブロックチェーンノード運用ガイド

27年間の会社員キャリアを経て、ゼロからブロックチェーンインフラエンジニアへ挑戦した実録手順書です。
コマンドひとつひとつに「なぜこれを打つのか」を添え、失敗した経験も学びとして記録しています。

## このリポジトリを使う方へ

### CLAUDE.md について

各ディレクトリに `CLAUDE.md`（執筆ルール・変更禁止事項）が存在しますが、
`.gitignore` により Git 管理対象外としています。

> 💡 **なぜ管理対象外にしているか：**
> `CLAUDE.md` には執筆方針・内部ルール・環境固有の設定が含まれます。
> 公開リポジトリに含める必要がないため、`.gitignore` で除外しています。
> クローン後に `CLAUDE.md` が存在しなくても、手順書の内容には影響しません。

### .gitignore の確認

```bash
# CLAUDE.md が除外設定されているか確認
cat .gitignore | grep CLAUDE
# → **/CLAUDE.md と表示されれば正常
```
