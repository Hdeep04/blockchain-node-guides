# blockchain-node-guides

## 概要
ゼロからブロックチェーンインフラに挑戦した実録手順書です。
コマンドひとつひとつに「なぜこれを打つのか」を添え、失敗した経験も学びとして記録しています。

## 構成

| フォルダ | 内容 | 状態 |
|---|---|---|
| ethereum/ | Ethereumバリデータ構築 3部作 | ✅ 公開中 |
| cardano/  | Cardano SPO構築手順書 | 🔜 準備中 |
| bitcoin/  | Bitcoinフルノード構築手順書 | 🔜 準備中 |

## Ethereum 3部作

| 部 | テーマ |
|---|---|
| 第1部・検証編 | VirtualBox VMでLido CSMバリデータを検証する |
| 第2部・ベアメタル編 | 鍵を専用物理PCへ引っ越し → 10鍵に増設 → 長期安定運用へ |
| 第3部・分散化(SSV)編 | 自サーバーのリソースを活かしSSVノードも兼業（二毛作） |

## 対象読者
- ITの基礎はあるが、ブロックチェーンは初めての方
- ホームステーキングに挑戦したい方
- キャリアチェンジとしてブロックチェーンインフラに興味がある方

## 特徴
- コマンドに「なぜ打つのか」の理由を添える
- ログや表示の読み方を初心者向けに解説
- 失敗した経験も学びとして正直に記録
- 機密情報（秘密鍵・パスワード等）は一切含まない公開版

## 注意事項
本書は公開用手順書です。
アドレス類はプレースホルダ（`<...>`）に置き換えてあります。
実際の値はご自身の環境に合わせて置き換えてください。
