## 対象読者
- ITの基礎はあるが、ブロックチェーンは初めての方
- ホームステーキングに挑戦したい方
- キャリアチェンジとしてブロックチェーンインフラに興味がある方

## 特徴
- コマンドに「なぜ打つのか」の理由を添える
- ログや表示の読み方を初心者向けに解説
- 失敗した経験も学びとして正直に記録
- 機密情報（秘密鍵・パスワード等）は一切含まない公開版
